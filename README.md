# 🚀 Projeto Simulação de Demanda Real

Este projeto foi desenvolvido com o objetivo de simular um cenário real de mercado, onde um cliente descreve suas necessidades e o desenvolvedor transforma essas informações em um site funcional.

A proposta foi treinar a interpretação de requisitos e a construção de soluções digitais baseadas em uma descrição fornecida.

---

## 📌 Sobre o Projeto

Neste projeto, simulei um cliente utilizando o ChatGPT, que descreveu como queria o site:

- Estrutura das seções
- Textos
- Serviços oferecidos
- Layout desejado
- Estilo visual

A partir dessas informações, desenvolvi todo o site utilizando:

- HTML
- CSS

O objetivo foi praticar o fluxo real de trabalho utilizado por empresas, onde o programador recebe um documento (como Word ou briefing) com as especificações e deve desenvolver conforme solicitado.

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido como preparação para o mercado de trabalho.

Recentemente participei de um processo seletivo onde a empresa trabalha com demandas reais enviadas pelos clientes, geralmente descritas em documentos detalhados.

O desenvolvedor deve:

- Interpretar os requisitos
- Seguir especificações técnicas
- Utilizar a linguagem definida pela empresa
- Entregar a solução conforme solicitado

Este projeto simula exatamente esse fluxo.

---

## 🧠 Uso de Inteligência Artificial

A Inteligência Artificial foi utilizada como ferramenta de apoio para:

- Simular o papel do cliente
- Organizar requisitos
- Melhorar estrutura de código
- Aplicar boas práticas

A IA foi utilizada como suporte estratégico, não como substituição do desenvolvimento.
